# EXP-3-PROMPT-ENGINEERING-

## Aim: 
Evaluation of 2024 Prompting Tools Across Diverse AI Platforms: 
ChatGPT, Claude, Bard, Cohere Command, and Meta
Experiment:
Within a specific use case (e.g., summarizing text, answering technical questions), compare the performance, user experience, and response quality of prompting tools across these different AI platforms.

## Algorithm:
Based on the provided technical analysis, the operational algorithms for each platform are categorized by their specific approach to data processing and reasoning logic.

### **ChatGPT (OpenAI)**
**Algorithm: System-2 Hierarchical Deduction**
* **Core Logic:** The algorithm focuses on multi-step logical deduction
* **Operational Method:** It employs a "System-2" thinking process specifically designed to reduce logical "drift" during extended technical explanations
* **Structuring Logic:** The model defaults to a highly structured hierarchy, prioritizing the use of headers (H1, H2) and bulleted lists

### **Claude (Anthropic)**
**Algorithm: Nuanced Constraint Satisfaction**
* **Core Logic:** This algorithm prioritizes nuanced communication and objective trade-off analysis over generic descriptions
* **Instruction Following:** It utilizes a high-fidelity constraint engine, achieving a 95% accuracy rate in following negative constraints (e.g., "Do not mention X")
* **Density Logic:** The algorithm is optimized for a high information-to-word ratio, intentionally stripping out introductory and concluding "filler" content

### **Gemini (Google)**
**Algorithm: Massive-Scale In-Context Learning (ICL)**
* **Core Logic:** Instead of relying solely on RAG (Retrieval-Augmented Generation), this algorithm leverages a 2-million-token window to perform "In-Context Learning"
* **Processing Method:** It allows for the simultaneous ingestion of massive datasets (such as 20+ whitepapers or hour-long videos) to identify commonalities across the entire corpus
* **Cross-Referencing:** The logic is designed to create comprehensive indices with functional cross-references between disparate parts of a large document

### **Cohere Command R+**
**Algorithm: Noise-Reduction Retrieval (RAG)**
* **Core Logic:** This algorithm is specifically tuned for "noise reduction" within private data environments
* **Groundedness Logic:** It prioritizes "anchoring" responses strictly to provided text to ensure maximum accuracy in enterprise settings

### **Meta (Llama)**
**Algorithm: Zero-Latency Iterative Feedback**
* **Core Logic:** The algorithm is optimized for maximum throughput and speed, delivering over 100 tokens per second
* **Operational Method:** It is designed for "rapid-fire troubleshooting" and iterative loops, particularly when hosted on local infrastructure to eliminate "typing delays"
